# Operation Guide

The AI Worker's Quick Start Operation Guide now includes three paths:

- **Teleoperation**: Control the AI Worker with the Leader–Follower system. Covers setup, launch options, trigger safety behavior, and joystick mappings for BG2 and SG2.
- **VR Teleoperation**: Control the AI Worker with the Leader–Follower system using VR. Covers setup, launch options, trigger safety behavior, and joystick mappings for BG2 and SG2.
- **Navigation**: Run the Nav2 stack for autonomous movement, including SLAM launch and map saving.
