# Setup Overview for AI Worker

This page summarizes the overall AI Worker setup flow. The detailed instructions are organized into separate hardware and software guides so you can move through the setup in order.

## Choose a Guide

- [Hardware Setup](/ai_worker/setup_guide_hardware_ai_worker): Power on the robot, identify the main ports, and release the initial safety lock with the Remote E-STOP.
- [Software Setup](/ai_worker/setup_guide_software_ai_worker): Connect to the robot PC, prepare the development environment on a user PC, and manage the Docker-based workflow.

## Recommended Order

1. Complete the [Hardware Setup](/ai_worker/setup_guide_hardware_ai_worker) steps for your robot model.
2. Continue with the [Software Setup](/ai_worker/setup_guide_software_ai_worker) guide to connect and prepare your development environment.
3. After setup is complete, proceed to the [Operation Guide](/ai_worker/operation_ai_worker).

## Notes

::: info
The robot PC inside AI Worker is pre-configured. In most cases, the software guide is used to set up a separate `user PC` for development and remote access.
:::
