# Technical Story

Hey there! Welcome to the **ROBOTIS Hand Technical Story** section.

ROBOTIS Hand is designed to explore dexterous manipulation, tactile sensing, and feedback-based grasping with a compact robotic hand platform. It provides a practical way to understand how finger mechanisms, sensors, and controllers work together to interact with real objects.

Beyond simple open-and-close motion, the hand can use tactile feedback to detect contact, maintain grasp force, and adapt its motion based on where the object touches the fingertip. These technical stories introduce the control methods and system design behind that behavior.
