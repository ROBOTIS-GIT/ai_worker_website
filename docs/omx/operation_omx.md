# Operation Guide

The OMX Quick Start Operation Guide now includes two paths:

- **ROS 2**: Bring up OMX with the ROS 2-based control stack for teleoperation, planning, and controller operation.
- **Motion controller**: Run the motion controller for task-space tracking, interactive-marker control, and `movej`/`movel` command-based operation.

Use the dedicated pages below to get started quickly:

- [ROS 2](/omx/operation_ros2_omx)
- [Motion controller](/omx/operation_motion_controller_omx)
