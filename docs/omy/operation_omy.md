# Operation Guide

The OMT Quick Start Operation Guide now includes two paths:

- **Teleoperation**: Bring up OMY with the Leader-Follower system for synchronized teleoperation and basic follower operation.
- **Motion controller**: Run the motion controller for task-space tracking, interactive-marker control, and `movej`/`movel` command-based operation.

Use the dedicated pages below to get started quickly:

- [Teleoperation](/omy/operation_teleoperation_omy)
- [Motion controller](/omy/operation_motion_controller_omy)
